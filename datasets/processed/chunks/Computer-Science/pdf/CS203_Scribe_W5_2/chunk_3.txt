Pseudocode for Top-Down Memoization: MEMOIZED-CUT-ROD(p, n) 1 let r[0...n] be a new array 2 for i = 0 to n 3 r[i] = -1 // Initialize 4 return MEMOIZED-CUT-ROD-AUX(p, n, r) 5.2 Bottom-Up Dynamic Programming This approach solves subproblems in increasing order. Pseudocode for Bottom-Up Approach: BOTTOM-UP-CUT-ROD(p, n) 1 let r[0...n] be a new array 2 r[0] = 0 3 for j = 1 to n 4 q = -1 5 for i = 1 to j 6 q = max(q, p[i] + r[j - i]) 7 r[j] = q 8 return r[n] 5.3 Comparison of Approaches Both approaches have the same time complexity of O(n2). 6 EXTENDED-BOTTOM-UP-CUT-ROD This algorithm computes both the maximum revenue and the first piece of length that leads to that revenue. Pseudocode: EXTENDED-BOTTOM-UP-CUT-ROD(p, n) 1. let r[0..n] and s[0..n] be new arrays 2. r[0] = 0 3. for j = 1 to n4. q = - 5. for i = 1 to j 6. if q < p[i] + r[j - i] 7. q = p[i] + r[j - i] 8. s[j] = i 9. r[j] = q 10. return r and s 6.1 Time Complexity: Both EXTENDED-BOTTOM-UP-CUT-ROD and PRINT-CUT-ROD-SOLUTION have a time complexity of O(n2). 7 Matrix-Chain Multiplication using Dynamic Pro- gramming 7.1 Problem Overview Given a chain of n matrices, the goal is to determine the most efficient way to multiply them. 7.2 Example For matrices A1 (10x20), A2 (20x30), etc., we find the optimal order of multiplication.
