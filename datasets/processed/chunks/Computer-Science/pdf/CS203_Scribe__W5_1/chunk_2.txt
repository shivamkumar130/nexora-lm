Combine In quicksort, no explicit combination step is required because the array is already sorted once the recursive calls return. The pivot naturally divides the sorted parts, so the entire array A[p... r] is now sorted. 2. Quicksort Implementation QUICKSORT(A, p, r) 1 if p < r 2 q = PARTITION(A, p, r) 3 QUICKSORT(A, p, q - 1) 4 QUICKSORT(A, q + 1, r) To sort an entire array A, the initial call is QUICKSORT(A, 1, A.length()). 2.1 Partitioning the Array The key to the algorithm is the PARTITION procedure, which rearranges the subarray A[p... r] in place. PARTITION(A, p, r) 1 x = A[r] # Choose the pivot as the last element 2 i = p - 1 # i is the boundary of the "less than or equal to pivot" partition 3 for j in range(p, r): 4 if A[j] <= x: 5 i += 1 6 A[i], A[j] = A[j], A[i] # Swap elements to move them to the left partition 7 A[i + 1], A[r] = A[r], A[i + 1] # Place the pivot in its correct position 8 return i + 1 # Return the pivot index The partition procedure is the key part of quicksort. It rearranges the elements of the subarray A[p... r] such that elements smaller than the pivot are placed on one side and those larger on the other.
