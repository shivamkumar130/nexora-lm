Problem 3 -- Perfect secrecy for (C=M\oplus K), with (M\in{0,1}^3), (K) uniform in ({0,1}^3) We must show the one-time-pad style XOR scheme with key uniformly random of same length gives perfect secrecy. \Pr[M=m\mid C=c] =\Pr[M=m]. Here (C = M \oplus K). For any fixed ciphertext (c) and any plaintext (m), there is exactly one key (k = c \oplus m) that produces (c) from (m). Since the key (K) is chosen uniformly from the 8 possibilities, (\Pr[K = c\oplus m] = 1/8). Thus \Pr[C=c\mid M=m] =\Pr[K = c\oplus m] = 1/8. By Bayes (or symmetry) this shows that (\Pr[M=m \mid C=c]) equals (\Pr[M=m]) (since the likelihood is the same for all plaintexts). Therefore the scheme gives perfect secrecy. key producing (c); hence observed (c) gives no advantage to an attacker - all plaintexts equally likely.)
