The multiplicative inverse map (x\mapsto x^{-1}) (in GF(2^8)) is nonlinear: it does not satisfy (S(x\oplus y)=S(x)\oplus S(y)) in general. A simple counterexample using known S-box values:. AES S-box value (S(0)=0x63) AES S-box value (S(1)=0x7C). Now (1\oplus1=0). If the S-box were linear over XOR, we would have (S(1\oplus1)=S(1)\oplus S(1)). But (S(0)=0x63), whereas (S(1)\oplus S(1)=0x7C\oplus0x7C=0). These are different, so the S-box is not linear. (More broadly: multiplicative inversion is not a linear map over the vector space GF(2)^8.)
