Data Preprocessing and Window Based Segmentation Noise and unwanted fluctuations are present in raw input data because of which preprocessing is done. First, IQR method was used to remove outliers. Then we apply low- pass filter to make signal smooth. Filtered voltage signal was generated which was further used in preprocessing. This preprocessing step was important to improve accuracy of model. Continuous breathing signal is divided into small segments on basis of sliding-window approach. One fixed window size is selected in which consecutive samples were included. Each window represent a breathing segment. Overlapping windows were used so that breathing transition was captured properly. For each window a label is assigned using majority - label approach. C. Feature Extraction From each segmented window statistical features were extracted. These features summarize the breathing sig- nal behaviour. Extracted features are standard devia- tion,mean,maximum value,minimum value,amplitude,mean gradient and variance. These features captures the signal amplitude and variation which help in differentiating the breathing patterns. D. Dataset Splitting and Feature Normalization Extracted feature dataset was divided into training and test set [37]. Training set was used to train the model and test set was used for evaluation. Stratified splitting was used to keep class distribution balanced. Features values were standardized to improve performance of model. Feature Scaling (Standardizaton) was applied to get mean as 0 and standard deviation as 1. This step is useful for model stability and better convergence. E.
