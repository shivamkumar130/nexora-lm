Scalability & Performance Challenges Real-time rendering and interaction with large-scale point cloud data is computationally expensive and poorly supported in immersive systems. These issues significantly limit the practical use of digital twins in smart manufacturing, infrastructure monitoring, smart cities, and cyber-physical systems. 3. Problem Statement Traditional digital twin systems suffer from: Limited spatial understanding due to non-immersive visualization Weak integration between geometric data and semantic knowledge Difficulty handling large-scale 3D point cloud data in real time. Lack of intuitive interfaces for querying and reasoning over system knowledge. As a result, digital twins often function as passive visual replicas rather than intelligent, interactive systems.. 4. Proposed Solution. This project introduces an immersive, intelligence-driven digital twin platform that: Represents real-world geometry using real-time 3D point clouds. Encodes system knowledge using a Knowledge Graph Synchronizes geometry, semantics, and live sensor data. Provides VR/AR-based immersive interaction for exploration, analysis, and. decision-making The proposed solution bridges the gap between physical realism, semantic intelligence, and human-centered interaction. 5. High-Level System Architecture The system follows a five-layer architecture, flowing from physical data acquisition to immersive user interaction: 1. Physical World & Data Sources 2. 3D Point Cloud Processing Layer 3. Knowledge Graph & Reasoning Layer 4. Digital Twin Core Engine 5. Immersive VR/AR Interaction Layer 6.
