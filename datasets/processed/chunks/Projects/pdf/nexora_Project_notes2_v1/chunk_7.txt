This project proposes a scaled-down but industry-relevant solution that leverages Computer Vision and Deep Learning (YOLOv8) to automate retail tasks such as product recognition, billing, and inventory monitoring. The system uses a standard webcam and software pipeline without requiring expensive loT hardware, making it both cost-effective and scalable. 2. Problem Statement: Traditional retail systems face challenges:. Long checkout queues. Manual billing and human errors.. Inefficient inventory management.. High infrastructure costs for automation. Research Question: Can we design a cost-effective Al-driven retail automation system that detects products, tracks customer actions, and generates bills in real-time without requiring heavy IoT infrastructure? 3. Objectives Automated Checkout: Detect products picked by customers and generate bills in real time. Inventory Monitoring: Use shelf-camera simulation to monitor stock levels and alert when restocking is needed. webcam and deep learning models.. Dashboard Integration: Provide a simple, interactive web dashboard for billing and inventory visualization. 4. Methodology: Step 1: Data Collection & Training. Capture images/videos of 3-5 retail products (e.g., chips, bottles, milk packets). Fine-tune YOLOv8 for product detection. Step 2: Real-Time Product Detection Use YOLOv8 to detect products through webcam feed. Each detected product is mapped to a price database.. Step 3: Shopping Cart & Billing System Track products when picked. Update cart dynamically and compute bill Step 4: Inventory Simulation Camera facing shelf -> detect and count product units. Alert if stock < threshold. Step 5: Dashboard. Use Streamlit/Flask to display: Current cart items + total bill. Inventory stock status. 5. Tools & Technologies: Deep Learning Frameworks: PyTorch, Ultralytics YOLOv8. Programming Language: Python. Web Framework: Streamlit/Flask. Database: SQLite/CSV for product-price mapping.. Hardware Requirement: Laptop with webcam (no loT mandatory). 6. Expected Outcomes: A working prototype that detects products in real time.. Automatic cart generation and bill calculation.. Inventory monitoring system via shelf camera simulation..
