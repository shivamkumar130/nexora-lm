Measurement Measurement is the process of comparing an unknown physical quantity with a standard quantity of the same nature. Examples: Measuring length using a meter scale. Measuring mass using a balance. Measuring time using a clock. The result of a measurement is expressed as: Measured Quantity = Numerical Value × Unit Example: Length = 5 m where: 5 = numerical value m = unit 2. Need for Measurement Measurement is necessary because: It allows comparison of physical quantities. It helps perform scientific experiments. It enables mathematical calculations. It provides quantitative information about physical phenomena. 3. Accuracy and Precision Accuracy Accuracy refers to how close a measured value is to the true value. Example: True value = 10.0 cm Measured value = 9.98 cm The measurement is highly accurate. Precision Precision refers to the degree of agreement among repeated measurements. Example: 10.01 cm 10.02 cm 1. 2. 3. 4. 10.01 cm 10.02 cm These readings are highly precise. Difference Between Accuracy and Precision Accuracy Precision Near true value Near each other Related to correctness Related to consistency Affected by systematic errorsAffected by random errors 4. Error in Measurement Every measurement contains some uncertainty. The difference between the measured value and true value is called error. Error = True Value − Measured Value Symbolically, E = T − M where: E = Error T = True Value M = Measured Value 5. Causes of Errors Errors arise due to: Instrument defects Observer mistakes Environmental conditions Experimental limitations 6.
