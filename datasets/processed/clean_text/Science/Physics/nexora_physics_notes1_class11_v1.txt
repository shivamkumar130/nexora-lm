UNIT–1: UNITS AND DIMENSIONS
1. Physical Quantity
Definition
A physical quantity is any quantity that can be measured.
Examples:
Length
Mass
Time
Temperature
Force
Energy
Physical quantities are divided into two categories:
(A) Fundamental Quantities
These quantities are independent and do not depend on any other physical quantity.
Examples: 1. Length 2. Mass 3. Time 4. Temperature 5. Electric Current 6. Luminous Intensity 7. Amount of
Substance
(B) Derived Quantities
These quantities depend upon one or more fundamental quantities.
Examples:
Area
Area = Length × Breadth
Dimension: [L²]
Speed
Speed = Distance / Time
Dimension: [LT⁻¹]
Volume
Volume = Length × Breadth × Height
Dimension: [L³]
2. Unit
Definition
A unit is the physical standard by which a physical quantity is measured.
Example:
Length → metre
Mass → kilogram
Time → second
Units are classified as:
(A) Fundamental Units
Units corresponding to fundamental quantities.
Examples:
metre (m)
kilogram (kg)
second (s)
(B) Derived Units
Units corresponding to derived quantities.
Examples:
Newton
Joule
Pascal
Watt
3. Systems of Units
Different systems of units have been used throughout history.

4. Supplementary Units
Plane Angle
SI Unit: Radian
Definition:
The angle subtended at the centre of a circle by an arc whose length equals the radius of the circle is called
one radian.
Formula:
θ = l/r
Where:
l = arc length
r = radius
Unit: Radian (rad)
Solid Angle
SI Unit: Steradian
Formula:
Ω = S/r²
Where:
S = surface area
r = radius
Unit: Steradian (sr)

LIMITATIONS OF DIMENSIONAL ANALYSIS
Although dimensional analysis is a powerful tool, it has some limitations.
1. Cannot Determine Numerical Constants
Dimensional analysis cannot determine numerical constants such as:
1/2
Example:
Area of circle
A = πr²
Using dimensions we can find that area is proportional to r² but cannot determine the value of π.
2. Cannot Distinguish Between Similar Quantities
Many physical quantities possess the same dimensional formula.
Examples:
Work
Dimension:
[ML²T⁻²]
Energy
Dimension:
[ML²T⁻²]
Torque
Dimension:
[ML²T⁻²]
All have the same dimensions but different physical meanings.
3. Cannot Derive Trigonometric Functions
Dimensional analysis cannot establish equations involving:
sinθ
cosθ
tanθ
logarithms
exponential functions
Example:
y = A sin(ωt)
cannot be derived completely using dimensions.
4. Cannot Check Vector or Scalar Nature
Dimensions only provide magnitude information.
They do not indicate whether a quantity is:
Scalar
Vector
Example:
Work and Torque possess identical dimensions but differ in nature.

ERRORS IN MEASUREMENT
Chapter Overview
Measurement is one of the most fundamental concepts in Physics. Every experiment, observation, and
calculation depends upon accurate measurements. Since no measurement is perfectly exact, the study of
errors becomes essential.
This chapter discusses:
Measurement
Accuracy and Precision
Errors in Measurement
Types of Errors
Absolute Error
Mean Absolute Error
Relative (Fractional) Error
Percentage Error
Error Propagation
Numerical Problems
1. Measurement
Measurement is the process of comparing an unknown physical quantity with a standard quantity of the
same nature.
Examples:
Measuring length using a meter scale.
Measuring mass using a balance.
Measuring time using a clock.
The result of a measurement is expressed as:
Measured Quantity = Numerical Value × Unit
Example:
Length = 5 m
where:
5 = numerical value
m = unit
2. Need for Measurement
Measurement is necessary because:
It allows comparison of physical quantities.
It helps perform scientific experiments.
It enables mathematical calculations.
It provides quantitative information about physical phenomena.
3. Accuracy and Precision
Accuracy
Accuracy refers to how close a measured value is to the true value.
Example:
True value = 10.0 cm
Measured value = 9.98 cm
The measurement is highly accurate.
Precision
Precision refers to the degree of agreement among repeated measurements.
Example:
10.01 cm
10.02 cm

4. Error in Measurement
Every measurement contains some uncertainty.
The difference between the measured value and true value is called error.
Error = True Value − Measured Value
Symbolically,
E = T − M
where:
E = Error
T = True Value
M = Measured Value
5. Causes of Errors
Errors arise due to:
Instrument defects
Observer mistakes
Environmental conditions
Experimental limitations
6. Classification of Errors
Errors are mainly classified into:
Systematic Errors
Random Errors
7. Systematic Errors
Systematic errors are errors whose causes are known and can usually be corrected.
These errors occur in the same direction repeatedly.
Characteristics:
Predictable
Reproducible
Can be reduced
Types of Systematic Errors
A. Instrumental Errors
Errors caused by imperfections in measuring instruments.
Examples:
Defective scale
Incorrect calibration
Zero error
B. Personal Errors
Errors caused by the observer.

Random errors occur due to unknown and unpredictable causes.
Characteristics:
Irregular
Unpredictable

9. Mean (True) Value
Suppose a quantity is measured n times.
Observations:
a₁, a₂, a₃,...., aₙ
The mean value is:
aₘ = (a₁ + a₂ + a₃ +... + aₙ)/n
This mean value is considered the most probable value of the measurement.
Example
Observations:
10 cm, 11 cm, 9 cm, 10 cm
Mean value:
aₘ = (10 + 11 + 9 + 10)/4
aₘ = 40/4
aₘ = 10 cm
10. Absolute Error
The absolute error for each observation is the numerical difference between the mean value and the
observed value.
For first observation:
Δa₁ = |aₘ − a₁|
For second observation:
Δa₂ = |aₘ − a₂|
Similarly,
Δaₙ = |aₘ − aₙ|
Example
Observations:
10 cm, 11 cm, 9 cm, 10 cm
Mean value = 10 cm
Absolute errors:
|10 − 10| = 0
|10 − 11| = 1
|10 − 9| = 1
|10 − 10| = 0
11. Mean Absolute Error
The arithmetic mean of all absolute errors is called mean absolute error.
Formula:
Δa = (|Δa₁| + |Δa₂| + |Δa₃| +... + |Δaₙ|)/n
Example
Absolute errors:
0, 1, 1, 0
Mean absolute error:
Δa = (0 + 1 + 1 + 0)/4
Δa = 2/4
Δa = 0.5 cm
Therefore,
Measurement = (10 ± 0.5) cm
12. Relative or Fractional Error
The ratio of mean absolute error to mean value is called fractional error.
Formula:
Fractional Error = Δa/aₘ
Properties:
No unit
Dimensionless quantity
Example
Mean value = 10 cm
Mean absolute error = 0.5 cm
Fractional error:
= 0.5/10
= 0.05
13. Percentage Error
Percentage error is obtained by multiplying fractional error by 100.
Formula:
Percentage Error = (Δa/aₘ) × 100
Example
Percentage Error
= (0.5/10) × 100
= 5%
Therefore:
Percentage Error = 5%
14. Significance of Error Analysis
Error analysis helps:
Determine reliability of measurements
Estimate uncertainty
Compare experimental results
Improve accuracy
Important Definitions
Measurement: Comparison of unknown quantity with a standard quantity.
Error: Difference between measured value and true value.
Absolute Error: Difference between mean value and measured value.
Mean Absolute Error: Average of absolute errors.
Fractional Error: Ratio of absolute error to mean value.
Percentage Error: Fractional error multiplied by 100.

SECTION A: IMPORTANT DEFINITIONS
Measurement
Measurement is the process of comparing an unknown quantity with a known standard quantity of the
same nature.
Error
The difference between true value and measured value is called error.
Error = True Value − Measured Value
Absolute Error
The magnitude of difference between mean value and measured value.
Mean Absolute Error
Arithmetic mean of all absolute errors.
Fractional Error
Ratio of absolute error to mean value.
Percentage Error

Question 1
Differentiate between Accuracy and Precision.
Accuracy
Closeness to true value.
Precision
Closeness among repeated observations.
Question 2
What are Systematic Errors?
Errors whose causes are known and can be corrected.
Examples:
Instrumental Error
Personal Error
Environmental Error
Question 3
What are Random Errors?
Errors caused due to unknown factors.
=r x +i^ y +j^ zk^
Characteristics:
Unpredictable
Irregular
Reduced by repeated observations
Question 4
State Triangle Law of Vector Addition.
If two vectors are represented by two sides of a triangle taken in order, the third side taken in reverse order
gives the resultant.
Question 5
State Parallelogram Law of Vector Addition.
If two vectors are represented by adjacent sides of a parallelogram, the diagonal through the common
point gives the resultant.
