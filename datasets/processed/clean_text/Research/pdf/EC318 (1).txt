Abstract—Respiration monitoring is crucial in the field of
human healthcare as it impacts various health conditions includ-
ing sleeping disorders like Apnea and respiratory diseases. Old
respiratory monitoring techniques like chest or abdomen belts
and nasal probes are uncomfortable to wear and expensive as
well. Thus, there is a requirement for a low cost and flexible
sensing system for continuous monitoring of breathing. This
paper describes the fabrication and working of a humidity-
based resistive sensor, designed to have an interdigitated electrode
structure. Breathing data is collected through an embedded
data acquisition system consisting of voltage divider circuitry,
LM358 unity-gain buffer and ESP32 microcontroller’s ADC.
Collected breathing signal data (a total of 30,000 samples) is
passed through a machine learning pipeline which includes data
labeling, noise removal using low pass filters, feature extraction,
stratified splitting of dataset and feature scaling. A random forest
classifier is trained and tested, resulting in an accuracy of 97.3%
approximately, in classifying into shallow, normal and deep
breathing patterns. Amplitude, maximum value and minimum
value features are found to be the most important features in
classification. The fabricated sensor showed a response time of 1.9
seconds and a hysteresis error of 10.17% in the characterization
plots. The proposed system in this study is effective in inputting
and analysing breathing signals using machine learning which
can further help in the diagnosis of sleep apnea and other related
health conditions.
Index Terms—respiration monitoring, resistive sensor, ESP32
microcontroller, Random Forest Classifier
I. INTRODUCTION
Respiration is a very important signal that indicates several
health conditions of a human being [1], [2]. If breathing is
different from normal, it can cause various health problems,
such as stress, Anxiety, sleep-related diseases [3], [4], and
lung-related problems. For example, when we breathe fast, it
shows stress or panic [3], [5]. And when we release air while
exhaling, respiratory issues show based on the temperature of
the air [6], [7]. Therefore, we should keep a constant check
on it to avoid anxiety and stress [3], [8].
In the past several years, due to the ways of living our
lives and problems occurring at the social level, many
health-related issues have started happening, like sleep-related
disorders [9].The most popular disorder among these is OSA
(Obstructive Sleep Apnea) [9], [10]. When we are in sleep
mode, it affects the breathing pattern [11]. if this condition
is occurs then it affect the human health condition such
as heart disease, high blood pressure, and sometimes even
sudden death if it is not identified in time [4]. When we
are asleep, there is a continuous lack of oxygen, which can
cause significant harm to the brain and heart [12]. Therefore,
breathing should be continuously and accurately monitored,
as it is very important for our daily healthy life.
Different sensing methods are used to monitor respiration
patterns, like temperature sensors and strain-based sensors
[13], [14]. Some traditional methods like polysomnography
are still used in hospitals to detect respiration patterns
[11]. To detect this, sensors and tubes are attached to the
body, which is quite uncomfortable [15]. Chest belts are
also used in a way to detect respiration monitoring. these
also affect body movement and create a lot of noise [15].
Other methods like camera-based systems and ultrasonic
sensors measure breathing patterns indirectly [16], [17] with
other sensor such as fiber-optic sensors are also used for
respiration monitoring [18]–[20]. Due to high sensitivity and
fast response,There are few sensors like humidity sensor is
used for monitoring the breathing pattern [21], [22]. When
a person breathes, i.e., inhales, the chest expands, and when
exhales, the chest contracts [15]. All these movements are
detected to generate an estimated breathing pattern.These
do not give accurate results of the breathing pattern [15],
[23].There are some devices that we use, like face masks or
nasal tubes. When these are used on the body to detect the
breathing pattern, they feel quite uncomfortable and, along
with that, they cannot be used on the body for a long time [15].
We have discussed earlier that traditional methods such
as polysomnography, nasal tubes, chest belts, camera
monitoring, and ultrasonic sensors were mostly used to detect
respiration patterns [11], [15]. However, these are quite
expensive, uncomfortable, and difficult to use for longer
durations [24], [25]. In many cases, due to body movements
[15], [26] and system limitations, inaccurate breathing
patterns are obtained. Based on above all issue faces, we
need to developed a wearable respiration monitoring system,
which is low in cost, easy to wear, i.e., comfortable, and easy
to use this system. it can detect breathing patterns easily and
accurately [27].
According to past research, Cao et al. [28] proposed a
low-cost flexible perforated respiratory sensor in their
research paper.They used platinum on poly-imide substrate.
When Sensor is Placed inside the nose,the temperature
change is detected upon breathing. the perforated sensor they
designed it enhance the gas connection, very less response
time, approx1.5%,sensitivity and enhance the monitoring
pattern like a normal, shallow and deep breathing.Another
researcher developed the chemiresistive sensor to detect
breathing patterns. This sensor using HKUST-1 metal-organic
framework and (MOF)MoS2 to detect sleep patterns in own
paper. it is detects every breathing pattern with very short
response time like a deep, slow, fast, hydrated breath. It is
stable over long term [29] without any decline in performance.
Based on past research, there are several limitations that
affect its results. We need a respiration monitoring system
that focuses not only on accuracy but also on simplicity, low
cost, and suitable for current times. Currently, many systems
rely on complex materials to measure respiration and focus
only on sensor design, not simplicity, and also on comfort.
They also don’t consider whether the system is comfortable
or not. They also don’t have data analysis reports [23], [30]
that can be used to analyze it and create accurate, simple,
and comfortable systems. All these factors create a gap in
signal acquisition and meaningful breathing patterns.
So we are proposing a flexible respiration monitoring
system which is a resistive humidity based sensor integrated
with an embedded data acquisition setup. This sensor detects
the breathing pattern as we inhale and exhale, causing a
change in humidity. Here,, the humidity concentration is
changing and hence the resistance is changing, due to which
the voltage is changing, i.e. it is converted into voltage
signals with the help of voltage divider circuit and it is further
processed with the help of signal conditioning techniques.
After this we are using ESP 32 micro controller to process
the signal so that data collection can be done in a simple and
easy way.
Also to increase the accuracy and usability of respiration
monitoring, machine learning models like supervised ml
models are being used and ml is being used for analysis
of respiration signal. First of all, the data obtained here
is preprocessed to remove noise and divide it into small
windows for segmentation. Statistical features are extracted
from each segment and used to train a random forest classifier
model. This model can easily identify the breathing pattern
whether it is shallow, normal or deep with high accuracy.
So our proposed system is combined with flexible humidity
sensor, embedded hardware and ml model. Based on this
it provides complete system for respiration monitoring.
It is low cost, easy to use, fast response and provides
reliable performance which allows us to do continuous health
monitoring and in future it can easily be used for wearable
respiration health care application so that we can prevent
respiration related disease.
II. EXPERIMENTALSETUP
A. Design and Fabrication of the Sensor
The flexible resistive humidity sensor [31] was designed in
the form of an interdigitated pattern of 27 interlocking fingers
arranged between 2 vertical electrode bars. Width of each
finger was set to 0.1 mm and the spacing between every two
consecutive fingers was set to 0.7 mm. The electrodes’ width
was also set to 0.1 mm and the spacing between them was
kept as 1 cm. The active sensing area of the designed sensor
is approximately 2cm 2 and its sensitivity to moisture has
been increased by populating the design with a high number
of electrical fingers. The max overall length of the humidity
sensor is 3.5 cm and overall width is 1.4 cm. The corners of the
rectangle shaped-sensor were kept rounded so as to ensure that
the corners don’t get stuck inside clothing when worn. Figure
1(a) displays the design of the resistive humdity sensor.
Fabrication process of this sensor involved printing the de-
signed humidity sensor on a polyimide based flexible substrate,
as shown in Figure 2. The flexible substrate was deposited
with a moisture sensitive material. The changes in humidity
concentrations directly cause variations in the resistance of the
sensor. The fully fabricated sensor has been shown in Figure
1(c). When a person inhales and exhales in front of the sensor
(Fig 1(b)), the humidity concentrations near the sensor vary
slightly, leading to measurable resistance changes. The design
of the sensor is optimal as moisture in the exhaled breath
gets settled down on the gaps between the fingers, changing
the conductivity of the air-substrate interface between the

C. Sensor Characterization
Various characterization tests were done to evaluate the
performance of the developed sensor [34]. Characterization
tests included the Hysteresis curve analysis, response time
measurement & temperature change effect on resistance. The
hysteresis curve plot (Figure 3) shows how the sensor’s
resistance changes when relative humidity concentration is
changed during breathing. Humidity concentrations are dis-
cretely changed from 0% to 100% during exhalation & 100%
to 0% during recovery. The up cycle shows the loading effect
(exhalation) of the sensor in which its resistance increases with
increasing humidity concentration. During exhaling, moisture
droplets deposit on the sensor’s surface, increasing the re-
sistance value, while the resistance value decreases during
the recovery phase (unloading effect) which represents the
down cycle in the graph. Calculation of hysteresis error for
this hysteresis curve gives a 10.17% error which shows good
repeatability of sensor but due to some minute moisture
droplets which get retained in the gaps on the sensor surface,
some lag has been observed.
The second characterization plot is about the measurement
of the fabricated sensor’s response time, as shown in Figure
4. The sensor’s response time [35] is defined as the amount of
time the sensor takes to reach 90% of its max voltage (V max)
from its baseline voltage value, during exhalation process.
Sensor was kept in a stable environment for a while to record
its baseline voltage value. Deep exhalation was then performed
starting atT start to record how the sensor’s resistance &
output voltage slowly increases to its maximum value. 90%
of the max voltage was calculated & the time (T 90%) was
Fig. 3. Hysteresis curve plot of the resistive sensor.
noted at which the sensor achieves 90%V max. And thus, the
calculated response time came out to be nearly 1.9 seconds
which is sufficiently fast to detect breathing cycles of a healthy
adult at resting condition (full breaths last for 4 to 5 seconds).
Fig. 4. Response time measurement of the humidity sensor.
The final characterization test was performed to observe
how the resistance of the sensor varies when temperature
of the surrounding environment is varied. To achieve this,
DHT11 temperature sensor [36] was used to measure the
temperature of the sensor’s surroundings while the sensor’s
resistance changes. The measured Resistance vs Temperature
plot in Figure 5 helps to understand that as temperature is
increased from 29°C to 33°C, the sensor’s resistance drops
from around 91kΩto 70kΩand after 33°C, its resistance
nearly fluctuates around the same level. The continuous de-
crease in sensor’s resistance intially can be explained from the
fact that with increase in temperature, kinetic energy of charge
carriers on the sensor’s surface increases, leading to decrease
in resistance. The resistance values stabilizing at 71kΩbut
with a few sudden peaks and drops is due to the sensor’s high
sensitivity to minor humidity changes happening because of
temperature change.

III. METHODOLOGY
A. Data Collection and Labeling
First the collection of voltage signal is done through res-
piration sensor. Output voltage changes according to sensor
breathing. Three different breathing patterns were recorded:
shallow breathing, normal breathing, deep breathing. For each
breathing type we have done multiple trials to ensure reliable
dataset. Sensor output is stored in format of time vs voltage
in csv files.
Collected dataset is labelled manually on basis of breathing
pattern. Each specific file is representing a specific breathing
class. Shallow breathing is assigned label 0, Normal breathing
is assigned label 1, Deep breathing is assigned label 2. All
three datasets were merged to form a single combined dataset.
Through this process, input-output pairs were prepared
B. Data Preprocessing and Window Based Segmentation
Noise and unwanted fluctuations are present in raw input
data because of which preprocessing is done. First, IQR
method was used to remove outliers. Then we apply low-
pass filter to make signal smooth. Filtered voltage signal
was generated which was further used in preprocessing. This
preprocessing step was important to improve accuracy of
model.
Continuous breathing signal is divided into small segments
on basis of sliding-window approach. One fixed window size
is selected in which consecutive samples were included. Each
window represent a breathing segment. Overlapping windows
were used so that breathing transition was captured properly.
For each window a label is assigned using majority - label
approach.
C. Feature Extraction
From each segmented window statistical features were
extracted. These features summarize the breathing sig-
nal behaviour. Extracted features are standard devia-
tion,mean,maximum value,minimum value,amplitude,mean
gradient and variance. These features captures the signal
amplitude and variation which help in differentiating the
breathing patterns.
D. Dataset Splitting and Feature Normalization
Extracted feature dataset was divided into training and test
set [37]. Training set was used to train the model and test set
was used for evaluation. Stratified splitting was used to keep
class distribution balanced.
Features values were standardized to improve performance
of model. Feature Scaling (Standardizaton) was applied to get
mean as 0 and standard deviation as 1. This step is useful for
model stability and better convergence.
E. Model Selection
Random Forest Classifier was used for breathing clas-
sification. Random Forest was choosen as it handles non-
linear data very well, overfitting reduces and also provides
feature importance. This algorithm works well for sensor-
based datasets.
Random Forest is one of the common machine learning
classifiers that merges predictions of several decision trees
in order to help in increasing classification performance and
decreasing overfitting [38]. Training of every decision tree
happens on randomly selected subset of training data and
feature set.We obtain final prediction class using maximum
votes of individual trees and express it as [39]:
ˆy= mode{T1(x), T2(x),..., Tn(x)}(2)
whereT i(x)isi th decision tree’s prediction,nis total
number of forest trees, andˆyis final predicted class.
F. Model Training and Evaluation
Training dataset was used to train Random Forest Classifier.
Multiple decision trees were generated and each tree train on
random subset of data. Final prediction was obtained from
majority voting mechanism. After training, model was ready
to classify between differnt breathing patterns.
Training model was evaluated on testing dataset. Predicted
labels were compared with actual labels. For performance
evaluation, different metrics were used like accuracy, preci-
sion, recall, F1-score and confusion matrix, etc. These metrics
measure the classification capability of model.

IV. RESULTS ANDDISCUSSION
We have collected samples over 5 minute interval over 10
people for training across normal, shallow, and deep breathing
in the ratio of 1:2:2 ratio as recommended during the data
collection process.
A. Breathing Signal Characterization
The respiration signals, which were collected from the sen-
sor were first analyzed to understand their actual behavior of
samples. The recorded voltage signal shows periodic variations
corresponding to breathing cycles. Figure 7 shows a sample
breathing waveform obtained from the sensor over time period.
The fluctuations in the signal represent the breathing changes
occurring during respiration.
Different breathing patterns shows us significant differences
in their signal characteristics. Shallow breathing typically
produces smaller amplitude variations, whereas Deep breath-
ing shows larger amplitudes over a period. While Normal
breathing lies between these two conditions.
B. Data Characterization
For machine learning based classification, several statistical
features were extracted from each signal segment units. These
features important properties of the respiration waveform and
helps us distinguish between different breathing patterns.

The extracted features were mean, standard dev, maxi-
mum, minimum, amplitude (difference between maximum
and minimum values), mean gradient, and signal variance.
These statistical descriptions summarize the distribution and
variations of respiration signal over time.
The collected dataset has 3 breathing categories: shallow,
normal, and deep breathing. Approximately 12000 samples
were collected for shallow breathing and deep breathing each,
while around 6000 samples were used for normal breathing.
This distribution follows a 1:2:2 ratio as recommended during
the data collection process.

G. Discussion
The results obtained from the experiments shows a strong
classification of breathing classes caputured from our cost
effective humidity sensor. Approx 97% was the accuracy of
trained Random Forest classifier, indicating that the selected
features provide meaningful information for identifying respi-
ration behaviour.
From the feature analysis, it can be observed that amplitude
features play an important role in separating breathing classes.
Deep breathing generally shows higher signal amplitudes,
which results in larger amplitude and variance values, while
shallow breathing tends to show small signals. Normal breath-
ing patterns lie between these two, which explains the slight
overlap seen in some feature distributions.
V. CONCLUSION
In our work, we developed a low-cost and flexible res-
piration monitoring system which can effectively detect and
analyze breathing pattern. This system is more used friendly
and practical as compared to traditional methods which are
very uncomfortable and expensive. We have used humidity
based resistive sensor which detect humidity change during
breathing and generate corresponding electric signals. For
processing of breathing data we collected, we have used
the machine learning pipeline that includes steps such as
preprocessing, feature normalization, etc. Random Forest Clas-
sifier has achieved 97.3% accuracy for classification task of
breathing patterns such as shallow, deep and normal which
show good reliability of system. This proposal system is
efficient, give fast response and provide accurate solution
for respiration monitoring. Practical applications of this can
be building healthcare applications especially the one which
can help in detecting sleep apnea disorders. Although results
we get are good but we need to consider some limitations
also. Experiments were done in real-world environment in
which some extra noise can come like movement of subject,
variation in sensor placement or it can be due to disturbances
in surroundings. In future, we can focus on working on larger
dataset and using advanced signal processing techniques to
make model match better with real - world conditions. Overall
it shows that combining feature normalization with machine
learning algorithm can give efficient solution for breathing
pattern classification.
